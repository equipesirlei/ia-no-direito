Configuração de Privacidade do
ChatGPT (OpenAI)

Como configurar sua conta na OpenAI para impedir que dados judiciais sejam usados no treinamento dos modelos, em conformidade com a Resolução CNJ nº 615/2025.

📜 Contexto Regulatório

A Resolução CNJ nº 615/2025 proíbe o uso de modelos de linguagem que utilizem dados do Poder Judiciário para treinamento próprio. O ChatGPT, por padrão, pode utilizar as conversas dos usuários para aprimorar seus modelos — a menos que o usuário desative essa opção expressamente.

⚠️ Atenção

Na configuração padrão, o ChatGPT armazena todas as conversas e pode utilizá-las para treinamento. Servidores e magistrados devem desativar essa funcionalidade antes de submeter qualquer dado processual.

Planos Gratuito vs. Pago: Diferenças na Proteção

⚠ Plano Gratuito (padrão)

  • Conversas podem ser usadas para treinamento dos modelos
  • Dados revisados por equipes da OpenAI para segurança
  • Histórico de conversas armazenado nos servidores
  • Menor controle sobre retenção de dados

✓ Plano Team / Enterprise (Recomendado)

  • Dados NÃO utilizados para treinamento por padrão
  • Conformidade com SOC 2 Type II
  • Controle administrativo sobre retenção de dados
  • Criptografia em trânsito e em repouso
  • Workspace isolado com políticas de segurança

💡 Alternativa para o Plano Gratuito

Mesmo no plano gratuito, é possível desativar o uso de dados para treinamento. A proteção não é tão robusta quanto no plano Enterprise, mas é suficiente para uso básico com dados não sensíveis. Para dados processuais, o plano Team ou Enterprise é o recomendado.

🔧 Passo a Passo: Desativar Treinamento

Método 1: Via Configurações do ChatGPT

Acesse as Configurações

No ChatGPT, clique no seu nome (canto inferior esquerdo) > Settings.

Vá em Data Controls

Na barra lateral, clique em Data Controls.

Desative "Improve the model for everyone"

Desative a opção que permite à OpenAI usar suas conversas para melhorar os modelos. A alteração é imediata.

Método 2: Via Portal de Privacidade da OpenAI

Acesse o Privacy Portal

Vá em privacy.openai.com e faça login com sua conta.

Solicite Exclusão de Dados

Você pode solicitar a exclusão de dados pessoais e do histórico de conversas diretamente pelo portal.

Chats Temporários

O ChatGPT oferece o modo "Temporary Chat", que não salva o histórico e não usa os dados para treinamento. Para ativá-lo, clique no menu do modelo (topo da conversa) e selecione "Temporary Chat". Ideal para consultas pontuais com dados sensíveis.
