Configuração de Privacidade do
Claude (Anthropic)

Como a Anthropic trata seus dados no Claude e quais configurações adotar para uso no âmbito do Poder Judiciário, em conformidade com a Resolução CNJ nº 615/2025.

📜 Política de Dados da Anthropic

A Anthropic, desenvolvedora do Claude, adota uma postura diferenciada em relação ao treinamento com dados de usuários. Por padrão, as conversas realizadas através da API e dos planos comerciais (Team e Enterprise) não são utilizadas para treinamento dos modelos.

No plano gratuito (claude.ai), a Anthropic pode utilizar conversas para melhoria dos modelos, mas oferece opção de exclusão (opt-out) fácil de ativar.

✅ Vantagem do Claude

Diferente de outros provedores, a Anthropic não treina com dados de usuários pagos por padrão. Nos planos Team e Enterprise, há compromisso contratual explícito de não utilização dos dados para treinamento, com período de retenção limitado e controles administrativos.

Planos e Níveis de Proteção

Plano Gratuito (claude.ai)

  • Conversas podem ser usadas para melhoria de modelos
  • Opção de opt-out disponível nas configurações
  • Dados retidos por até 90 dias após exclusão
  • Adequado para consultas sem dados sensíveis

✓ Plano Pro / Team / Enterprise (Recomendado)

  • Dados NÃO utilizados para treinamento por padrão
  • Compromisso contratual de não treinamento
  • Retenção de dados limitada (configurável no Enterprise)
  • Conformidade com SOC 2 Type II
  • Controles administrativos avançados (Team/Enterprise)

🔧 Passo a Passo: Configuração de Privacidade

Para o Plano Gratuito / Pro (claude.ai)

Acesse as Configurações

No claude.ai, clique no seu nome (canto inferior esquerdo) > Settings.

Desative o Treinamento com Dados

Na seção de privacidade, desative a opção "Allow Anthropic to use your conversations to improve Claude". Isso impede que suas conversas sejam usadas para treinamento.

Gerencie o Histórico

Você pode excluir conversas individuais ou todo o histórico a qualquer momento. Após a exclusão, os dados são removidos dos sistemas ativos em até 90 dias.

Para Claude Code (CLI) e API

API: Proteção Automática

Dados enviados via API nunca são usados para treinamento, independentemente do plano. A retenção é de no máximo 30 dias para monitoramento de segurança, sem uso para melhoria de modelos.

Claude Code: Configurações Locais

O Claude Code processa dados localmente na sua máquina. As interações com a API seguem as mesmas políticas de não treinamento. Dados do sistema de arquivos local não são transmitidos além do necessário para a consulta.

💡 Recomendação para o Judiciário

Para uso institucional com dados processuais, a combinação mais segura é: plano Pro ou Team + opt-out de treinamento ativado. Para uso via API (integrações automatizadas), a proteção já é nativa e não requer configuração adicional.
